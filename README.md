# 鲸舰电玩城管理系统

基于Vue3 + Element Plus的现代化电玩城管理系统

## 项目概述

鲸舰电玩城管理系统是一套完整的电玩城运营解决方案，涵盖日常运营、会员管理、设备监控、营销推广等核心功能。

## 技术栈

- **前端框架**: Vue 3.4.21
- **UI组件库**: Element Plus 2.6.1
- **路由管理**: Vue Router 4.3.0
- **状态管理**: Pinia 2.1.7
- **构建工具**: Vite 5.2.0
- **HTTP请求**: Axios 1.6.8

## 功能模块

### 核心业务模块
- **日常管理**: 日常运营记录、班次管理
- **套餐管理**: 游戏套餐配置、价格管理
- **会员管理**: 会员信息、积分、余额管理
- **商品管理**: 商品库存、销售管理
- **营销管理**: 营销活动、推广管理
- **设备管理**: 设备监控、状态管理
- **氛围管理**: 环境氛围配置
- **台球管理**: 台球厅业务管理
- **线上管理**: 线上业务管理
- **报表管理**: 数据报表分析

### 系统设置模块
- **员工权限**:
  - 员工管理：员工信息维护
  - 职位权限：权限等级配置
- **运营监控**:
  - 风控设置：风险控制配置
  - 异常记录：异常事件管理
  - 导出中心：数据导出功能
  - 系统日志：运行日志查看
  - 回收站：已删除数据管理
- **基础设置**:
  - 财务设置：财务参数配置
  - 消息管理：消息推送设置
  - 支付商户：支付渠道管理
- **更多设置**: 系统高级设置
- **数据管理**: 数据备份与恢复
- **开放平台**: API接口管理

## 系统特色

### 多级菜单系统
- 一级菜单：主要功能模块

## 安装和运行

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖
```bash
npm install
```

### 开发环境
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 开发说明

### 页面开发
- 所有页面组件位于 `src/views/` 目录
- 使用 Element Plus 组件库进行界面开发
- 遵循 Vue 3 Composition API 规范

### 样式规范
- 使用 SCSS 预处理器
- 采用 BEM 命名规范
- 响应式设计优先

### 路由管理
- 使用 Vue Router 进行路由管理
- 支持路由懒加载
- 配置路由元信息
